## 📖 사용 방법

### 기본 사용법

1. **할 일 추가**
   - 입력창에 할 일 입력
   - 원하는 색깔 태그 선택 (선택사항)
   - Enter 또는 "추가" 버튼 클릭

2. **할 일 완료**
   - 체크박스 클릭

3. **할 일 편집**
   - 텍스트 더블클릭
   - 또는 편집 버튼 클릭

4. **할 일 삭제**
   - 휴지통 아이콘 클릭

5. **태그 사용**
   - 입력창 아래 "태그:" 섹션에서 색상 선택
   - 선택된 태그는 테두리 강조
   - 할 일에 작은 색깔 점으로 표시

### 고급 기능

#### 검색
```
1. 검색창에 키워드 입력
2. 실시간으로 필터링됨
3. X 버튼으로 검색 초기화
```

#### 날짜 선택
```
1. 헤더의 캘린더 위젯 클릭
2. 원하는 날짜 선택
3. 해당 날짜의 할 일만 표시됨
4. "날짜 지우기"로 필터 해제
```

#### 테마 변경
```
1. 헤더의 태양/달 아이콘 클릭
2. 다크/라이트 모드 전환
3. 설정 자동 저장
```

## 🎨 주요 기능 상세

### 1. 할 일 관리
- **추가**: 입력 후 Enter 또는 버튼 클릭
- **완료**: 체크박스로 완료 상태 토글
- **편집**: 더블클릭 또는 편집 버튼
- **삭제**: 개별 삭제 또는 완료된 항목 일괄 삭제

### 2. 필터링 & 검색
- **상태 필터**: 전체/진행중/완료
- **텍스트 검색**: 실시간 검색
- **날짜 필터**: 특정 날짜의 할 일만 표시

### 3. 캘린더
- **현재 날짜**: 자동 업데이트
- **월간 뷰**: 이전/다음 달 이동
- **날짜 선택**: 클릭으로 필터링
- **할 일 표시**: 점으로 표시

### 4. 색깔 태그
- **5가지 색상**: 핑크, 블루, 그린, 피치, 퍼플
- **태그 선택**: 할 일 추가 시 선택
- **시각적 표시**: 색깔 점으로 표시
- **카테고리화**: 색상별로 할 일 구분

### 5. 데이터 저장
- **자동 저장**: 모든 변경사항 즉시 저장
- **테마 저장**: 선택한 테마 기억
- **LocalStorage**: 브라우저 로컬 저장소 사용
